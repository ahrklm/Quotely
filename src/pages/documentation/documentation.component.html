Quotely — Technical & Functional Guide

Internal System Documentation for Air France-KLM Project Management

01 Core Concept

Quotely is a high-fidelity internal utility designed to standardize the project estimation lifecycle. It bridges the gap between raw developer estimates and formal business proposals, replacing fragmented Excel files with a centralized, token-secured web platform.

02 Feature Deep-Dive

Smart Duplication

When duplicating a Quote or Template, the system performs a "Deep Copy". It maps new UUIDs for every Phase (Section) and Line Item, maintaining the original hierarchical relationships so that tasks remain assigned to their correct phases in the new copy.

EPFM Security Gate

External approvals via the Client Link require a mandatory 5-digit EPFM (External Project Finance Management) number. This acts as a digital signature, ensuring that only authorized stakeholders can transition a quote to the 'Approved' state.

03 The Workflow Engine

Draft / Internal:

The default state for new estimates. Full CRUD permissions for dev teams to define phases and tasks.

Shared (Pending Review):

A tokenized URL is generated. The client can view the estimate but cannot yet approve it until it is officially submitted by the creator.

Approved:

Locked state. Requires EPFM validation. Triggers a notification to finance (simulated via status update).

QuoteSection Title, sortOrder Groups items within a Quote
